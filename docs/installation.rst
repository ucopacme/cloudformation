============
Installation
============


Stable release
--------------

To install UCOP ACME python cloudformation helper aka make it rain, run this command in your terminal:

.. code-block:: console

    $ pip install ucop_acme_make_it_rain

This is the preferred method to install UCOP ACME python cloudformation helper aka make it rain, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for UCOP ACME python cloudformation helper aka make it rain can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/ericodell/ucop_acme_make_it_rain

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/ericodell/ucop_acme_make_it_rain/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/ericodell/ucop_acme_make_it_rain
.. _tarball: https://github.com/ericodell/ucop_acme_make_it_rain/tarball/master
